About

ParentOrb helps parents automate repetitive tasks, like being ready at set times, chore deadlines, and curfews.

Currently, parents interact with the app on Facebook Messenger, while their kids receive updates via text message. A mobile app is planned also.

Using the same artificial intelligence that powers Amazon's Alexa, ParentOrb is able to interact with both parents and kids in conversational english.

We are still in beta, try it out, we welcome your feedback.
